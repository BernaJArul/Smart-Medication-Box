# Smart-Medication-Box
Arduino-based smart medication reminder system for improving medication adherence and patient safety.

## Overview

The Smart Medication Box is an Arduino-based healthcare device designed to help users take their medications on time. The system uses a Real-Time Clock (RTC) module to track time accurately and provides timely reminders through a buzzer and LCD display. It is particularly beneficial for elderly individuals, patients with chronic illnesses, and those who need assistance in managing medication schedules.

## Objectives

- Provide timely medication reminders.
- Improve medication adherence and patient safety.
- Reduce dependency on memory and external assistance.
- Offer a simple, low-cost, and user-friendly healthcare solution.

## Working Principle

The system uses an Arduino Uno microcontroller integrated with a DS3231 Real-Time Clock (RTC) module. Users can set medication schedules using push buttons. At the programmed time, the system activates a buzzer and displays reminder messages on the LCD screen, prompting the user to take their medication.

## Results

The Smart Medication Box successfully provides reliable medication reminders with minimal delay. Testing demonstrated accurate scheduling, clear notifications, and ease of use. The system helps users maintain medication adherence and reduces the risk of missed doses.
